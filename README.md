## E-Commerce Backend

This folder contains the Mock Backend created for E-Commerce App. It contains different API end-points and database configuration for building E-Commerce; so that you can focus on building Frontend.

## Feature

- Product Listing
- Single Product Page
- Add to Wishlist 
- Add to Cart 
- 404 page
